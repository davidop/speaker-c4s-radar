## Task 07-01-02: Añadir endpoints /api/proposals en middleware local

**type:** implementation
**files:** demo/vite.config.js

**action:**
Agregar middleware para propuestas:
- `GET /api/proposals` -> lista
- `POST /api/proposals` -> crear (id incremental `p-###`)
- `PATCH /api/proposals/:id` -> actualizar
- `DELETE /api/proposals/:id` -> eliminar

Validaciones mínimas:
- `title` requerido en create/update
- `tags` array normalizado
- 404 para ids no encontrados

**verify:** Respuestas JSON y códigos HTTP esperados.
**acceptance_criteria:** CRUD propuestas disponible en modo `npm run dev`.
