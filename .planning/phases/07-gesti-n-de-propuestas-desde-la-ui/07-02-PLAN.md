## Task 07-02-02: Crear mini panel de gestión de propuestas

**type:** implementation
**files:** demo/src/main.js, demo/src/styles.css

**action:**
Añadir sección "Propuestas" con:
- lista de propuestas
- botón "Nueva propuesta"
- acción editar/borrar por propuesta

Operaciones:
- crear propuesta -> `POST /api/proposals`
- editar -> `PATCH /api/proposals/:id`
- borrar -> `DELETE /api/proposals/:id`

Regla de seguridad:
- Si una propuesta está vinculada a algún call, bloquear borrado con mensaje.

**verify:** CRUD propuestas usable desde UI.
**acceptance_criteria:** Gestión completa de propuestas desde UI sin tocar código.

</tasks>

</verification>
